# BeMoningNinja

BeMorninngNinja lets you wake up and/or motivate with your fav instagram influencer.

#### Stack:

Backend: 
1. Play Scala Application (`/app`)

FrontEnd: 
1. HTML/Javascript for Web (`/index.html`)
2. Alexa Skill Lambda (`/alexa`)
